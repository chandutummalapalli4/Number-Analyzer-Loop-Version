# 🔢 Number Analyzer (Loop Version)

## 📌 Project Description

The Number Analyzer is a Python program that takes a number as input and performs multiple digit-based operations using loops. It helps beginners understand how to work with loops, conditions, and mathematical operations on digits.

---

## 🚀 Features

- ✅ Sum of digits
- ✅ Product of digits
- ✅ Count of digits
- ✅ Reverse number
- ✅ Largest digit
- ✅ Smallest digit
- ✅ Even and Odd digit identification
- ✅ Palindrome check
- ✅ Armstrong number check

---

## 🧠 Concepts Used

- while loop
- if-else conditions
- modulus operator (%)
- integer division (//)
- arithmetic operations
- logical comparisons

---

## ⚙️ Logic Overview

```
Step 1: Take number input
Step 2: Extract digits using % 10
Step 3: Reduce number using // 10
Step 4: Update:
        - sum
        - product
        - reverse
        - max digit
        - min digit
        - count
        - even/odd classification
Step 5: Use copy of number for Armstrong check
Step 6: Compare reverse with original for palindrome
Step 7: Compare Armstrong result with original
```

## 💻 Sample Code Features

- Loop-based digit extraction
- Multiple calculations in single traversal
- Reuse of number copy for different logic blocks
